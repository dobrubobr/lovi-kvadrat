# 🎯 Лови квадрат

> Игра на реакцию и скорость с уникальными режимами, системой комбо, достижениями и таблицей рекордов

![Лицензия](https://img.shields.io/badge/license-MIT-green)
![Телефон](https://img.shields.io/badge/📱-адаптив-success)

## 🎮 Игровой процесс

Квадрат появляется в случайном месте игрового поля. Ваша задача — **кликать по нему как можно быстрее**. Каждый клик приносит очки, а квадрат мгновенно перемещается в новое место.

**Особенность:** квадрат сам автоматически прыгает каждые X секунд (зависит от выбранной сложности), так что нужно успевать!

## ✨ Особенности

### 🎚️ 5 уровней сложности
| Уровень | Прыжок | Время игры |
|---------|--------|------------|
| 🌱 Новичок | 3.5с | 45с |
| 🍃 Лёгкий | 2.0с | 35с |
| ⚡ Средний | 1.5с | 30с |
| 🔥 Тяжёлый | 0.7с | 20с |
| 💀 Эксперт | 0.2с | 12с |

### 🎲 3 дополнительных режима
- **♾️ Бесконечный** — играйте без ограничения по времени, но 3 промаха — и игра окончена
- **👾 Режим босса** — с каждым кликом квадрат уменьшается! Становится всё сложнее
- **👥 Мультиплеер** — 30 секунд на двоих. Зовите друзей и соревнуйтесь!

### 🔥 Система комбо
**3 клика подряд** → активирует режим **x2 очков** на 3 секунды!

### 🏆 Достижения
- 🎯 **Первый шаг** — набрать 10 очков за игру
- 👑 **Эксперт** — победить на уровне Эксперт 5 раз
- 💎 **Идеальная игра** — набрать 30 очков без использования комбо
- ⚡ **Скоростной демон** — сделать 50 кликов за одну игру
- 🔥 **Комбо-мастер** — активировать комбо 10 раз за игру

### 📊 Статистика
- Общее количество очков
- Всего сыграно игр
- Всего кликов
- Таблица рекордов (Топ-3 для каждого уровня)
- Все достижения с отслеживанием прогресса

### 🎨 Кастомизация
- **Выбор цвета квадрата** — любой цвет на ваш вкус
- Фон главного меню меняется под цвет выбранного уровня

### 📱 Адаптив
Игра полностью адаптирована для **телефонов и планшетов**:
- Отзывчивый интерфейс
- Поддержка touch-событий
- Адаптивное игровое поле
- Удобные крупные кнопки

## 🚀 Как играть

1. Выберите уровень сложности
2. (Опционально) Настройте цвет квадрата
3. Нажмите «Начать игру»
4. Кликайте по квадрату, набирайте очки
5. Ставьте рекорды и получайте достижения!

## 🛠️ Технологии

- HTML5
- CSS3 (адаптивная верстка, анимации)
- JavaScript (ванильный)
- LocalStorage (сохранение рекордов и статистики)
